INTERVIEW QUESTIONS ON CSS

what is css?why do we use it?

CSS stands for casecading style sheet.It used to give the styles to Html document or website or webpage.

In how many ways can we create css styling?What are they?

We can apply styles to the html document in 3 ways .
They are:
1.Inline css
2.Internal css
3.External css

Difference b/w 3 ways of css styling?

In Inline css we can give styles to the particular element by using style attribute.In internal css we can give styles to the particular element in style tag .In External css, we can create one external file and saved as filename.css.That file is attached to the html document by using link tag,after that we can apply styles to the particular element by using external css file.

What is specificity among the ways of css styling?

Inline css has high specificity when compared to internal and external .Internal css is high specificity when compared to external css.

What is the syntax of css?

Selector{
property:value;
}

What is a selector?

Selector is select the element what we want to style.

How many types of selectors?

They are :
1.Simple selector
2.Combinator Selector.
3.Pseudo-Class Selector
4.Pseudo-elements Selector
5.Attribute Selector.

What are the general selectors?

General Selector is also known as simple selector. It used to select the elements based on name,id,class.In which we have 5 types of selectors.
They are:
1.Class selector(It will accessed by .)
2.Id selector(It will accessed by #)

What is difference b/w the pseudo class vs pseudo element?

Pseduo classes are used to give styles based on their states(focusing,active) and it is denoted by single colon.Where as pseduo elements are used to give styles to the specific part of the element.It is denoted by double colon.

What is the rule set?

A CSS rule set defines how to style elements. It consists of:1. Selector: Specifies which HTML element(s) to style. 2.Declaration Block: Contains the styles to apply, made of:
Property: The aspect to style (e.g., color, font-size).
Value: The value for the property (e.g., red, 16px)

What is the difference b/w padding and margin?

Padding is used to give the space between content and border.Where as margin is used to give space outside of the border.

What is difference b/w outline and border?

An outline is a line that is drawn around elements, OUTSIDE the borders, to make the element "stand out".A border that goes around the padding and content.

What is the difference b/w display and float?

Display property is used to display the elements in webpage/website. float property is used to float the elements in webpage/website.

What is difference b/w the display:inline vs display:block vs display:inline-block?

display:inline property is used to convert block level element into inline level element.display:block property is used to convert inline level element into block level element Where as display:inline-block property is used to convert inline to block and here we can add margin top and bottom to that element.Generally to inline element margin is not worked if we use display:inline-block then only it will be worked.

what is difference b/w the visibility:hidden and display:none?

By using visibility:hidden we can hide the content and here it will space also. By using display:none we can remove the content from webpage and it will not give any space.

what is object-fit ? and in how many types can we fit the object?

object-fit is used to fit the content in container.We can fit the object in 4 types.
They are :
1.object-fit:contain(over all content is zoom-in)
2.object-fit:cover(content placed in center)
3.object-fit:fill (totally filled)
4.object-fit:none(heavily zoomed)

What is background property? what are the properties of background do you know?

Background property is used to give styles of the background.Here we have some background properties.Such as background-color,background-image,background-size,background-position,background-repeat.

what is the position property?

position property is used to positioning the elements in the document.

Types of position values we can give?

we have 5 types of position values.
They are:
1.position:static
2.position:fixed
3.position:sticky
4.position:absolute
5.position:relative

what is difference b/w the position:sticky and position:fixed?

position: sticky: Sticks to a position within its container when you scroll to it, then moves with the container.position: fixed: Stays in a fixed position on the screen, regardless of scrolling, and does not move with its parent.

what is difference b/w the position:relative and position:absolute?

position:relative:The element is positioned relative to its normal position.A relative position moves the element, but still consumes some space in the original position. position:absolute:the element is positioned absolutely to its first positioned parent.An absolute or fixed position moves the element, but does not consume any layout space.

What is overflowing in css?

overflow property is used to control the overflowed content(content,image).

What is z-index in css? default value of z-index?

To maintain the stack order of the element in vertical direction or vertical alignment of the element.Default value is Z-index:0.

What is a clear property?

The clear property in CSS is used to control the behavior of floating elements. It specifies whether an element can be next to floated elements or should be moved below them.

What are the css variables?

CSS variables are used to create and reuse through out the style sheet.It helps us to maintain the code in globally.

What is :root in css?

In CSS variables, :root is used to define global variables (custom properties) that can be accessed anywhere in the stylesheet or we can reused throught the stylesheet.

What are the font -properties?

What is a flex box in css?

flex is a single dimensional layout .We can align the elements in horizental direction or vertical direction.

What is a grid layout system?

Grid is a 2D layout and we can align the elements in horizental direction and vertical direction.

Difference b/w the flex vs grid?

Grid is a 2D layout and we can align the elements in horizental direction and vertical direction.It's default direction is column.flex is a single dimensional layout .We can align the elements in horizental direction or vertical direction.It's default direction is row.

What are media queries in css? why do we need to use them?

Every website is fited to any device(mobile,tablet,laptop,large laptop,4k screens) according to the device width ,then it is called media queries .It is a powerful tool to make a website responsive.

What are transitions in css?

Transition is allows you to change css property values smoothly over a specified durarion.

What are the transforms in css?

Transforms are used to change the form,appearance or structure of an element.It allows you to move,rotate,scale and skew elements.

How can you center a div?

We can center a div in the below ways.we can use display:grid,place-items:center for getting element as center.By using display:flex,justify-content:center,align-items:center we can make an element as center and we can use margin and padding also for this purpose.

what is difference b/w the transitions vs transforms vs animations?

Transition is allows you to change css property values smoothly over a specified durarion.Transforms are used to change the form,appearance or structure of an element.It allows you to move,rotate,scale and skew elements.Where as animations are gradually change from one style to another and we must specify some keyframes for the animation.Keyframes are used to control the motion of element.

What is a matrix in transforms?

In CSS transform, the matrix() function allows you to apply a 2D transformation by combining multiple transformations (like scaling, rotating, translating, and skewing) into a single function.

Difference b/w css and css3?

CSS: The original version, used for basic styling (colors, fonts, margins, etc.). CSS3: An updated version with new features like animations, transitions, gradients, media queries, and more. CSS3 is an enhanced version of CSS with advanced styling capabilities for modern web design.
